Determine the value of the angular frequency, $\omega$ in radians per second for a sine wave, which is a function of continuous time, $t\in\mathcal{R}$ with units of seconds, for the following parameters:
Sine wave frequency:$f=$Hz
\[\omega=2\pi f\]
\[\]
Provide the mathematical definition for a sine wave which is a function of continuous time, $t\in\mathcal{R}$ with units of seconds given the following parameter values:
Sine wave frequency:$f=$Hz
Amplitude:$A=$arbitrary units
Phase:$\phi=$radians
\[x(t)=A\sin(2\pi f t + \phi)\]
\[\]
Provide the mathematical expression for a digitally sampled sine wave, which is a function of a discrete time index $n\in\mathcal{N}$ with the following parameters:
Sine wave frequency:$f=$Hz
Amplitude:$A=$arbitrary units
Phase:$\phi=$radians
Sampling frequency:$f_s=$Hz
\[x[n]=A\sin\left(\frac{2\pi f n}{f_s} + \phi\right)\]
\[\]
Provide the mathematical expression for a digitally sampled sine wave, which is a function of a discrete time index $n\in\mathcal{N}$ with the following parameters:
Normalised digital frequency:$\Omega=$rad/sample
Amplitude:$A=$arbitrary units
Phase:$\phi=$radians
\[x[n]=A\sin\left(\Omega n + \phi\right)\]
Determine the analogue frequency in Hz of a signal given the following parameter values:
Normalised digital frequency:$\Omega=$rad/sample
Sampling frequency:$f_s=$Hz
\[\Omega=\frac{2\pi f}{f_s}\]
